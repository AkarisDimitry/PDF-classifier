# PDF-classifier
Este proyecto proporciona un script en Python para realizar clustering no supervisado en datos de texto a partir de archivos PDF y DOC. El script lee y preprocesa los datos de texto, los convierte en una matriz TF-IDF y luego realiza cuatro tipos diferentes de clustering: K-means, jer√°rquico, Latent Dirichlet Allocation (LDA) y DBSCAN.
